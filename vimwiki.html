什么是vimwiki

Vimwiki 是用vim来搭建的个人wiki。
特性：

  • 组织想法和日记
  • 管理待办事项列表
  • 写文档


按\ww打开wiki主页，默认位置为~/vimwiki/index.wiki。

vimwiki安装

.vimrc中需有如下设置： 

	set nocompatible
	filetype plugin on
	syntax on


本文的.vimrc设置： 

"""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""
" Owner:	Tiny - http://www.tinyxd.me
" Modified: 2013-01-24 22:11:09

" set runtimepath=~/vim,$VIMRUNTIME
" source ~/vim/vimrc

" Get out of VI's compatible mode..

"""""""""""""""""""""""""""""""""""""""
"vimwiki设置
"""""""""""""""""""""""""""""""""""""""
" 打开语法高亮
syntax enable
syntax on

" 打开对文件类型插件的支持
filetype indent on
filetype plugin on
filetype plugin indent on

" 去掉有关vi的一致性模式，避免以前版本的一些bug和局限
set nocompatible

" ######### VimWiki 写作助手 ######### "

" 使用鼠标映射
let g:vimwiki_use_mouse = 1

" 不要将驼峰式词组作为 Wiki 词条
let g:vimwiki_camel_case = 0

let g:vimwiki_list = [{
\ 'path': '/media/software/Dropbox/my_vimwiki_site/wiki',
   \ 'template_path': '~/.vim/autoload/vimwiki/',
    \ 'template_default': 'default',
    \ 'template_ext': '.tpl', 
\ 'path_html': '/media/software/Dropbox/my_vimwiki_site/html/',
\ 'auto_export': 1,},
\{
\'path': '/media/software/Dropbox/my_index_site/wiki',
   \ 'template_path': '~/.vim/autoload/vimwiki/',
    \ 'template_default': 'myindex',
    \ 'template_ext': '.tpl',
\ 'path_html': '/media/software/Dropbox/my_index_site/html/',
\ 'auto_export': 1,}]

常用快捷键

  • [count]<Leader>ww 在当前窗口打开第[count]个wiki首页
  • [count]<Leader>wt 在新标签页中打开第[count]个wiki索引文件
  • <Leader>ws 列出可选择的wiki列表
  • [count]<Leader>w<Leader>w 打开今天第[count]个wiki日记
  • [count]<Leader>w<Leader>t 在新标签页打开今天第[count]个wiki日记
  • [count]<Leader>wi 打开日记的首页
  • <Leader>wh 把当前wiki页面生成HTML--对应命令Vimwiki2HTML
  • <Leader>whh 把当前wiki页面转换为HTML并用浏览器打开
  • <Leader>w<Leader>i 更新diary主页（用于写完新的日志后，更新diary主页）
  • <Leader>wd 删除当前wiki页
  • <Leader>wr 重命名当前wiki页面

常用命令

  • :Vimwiki2HTML 将当前页面转为HTML
  • :VimwikiAll2HTML 将所有wiki页面转化为HTML
  • :VimwikiSearch /pattern/ 在当前wiki中搜索/parttern/::lopen显示所有匹配，:lnext显示下一个匹配，:lprevious显示上一个匹配
  • :VimwikiTable 创建表格::VimwikiTable cols rows创建cols行rows列的表格

wiki语法

  • 文本字型
    • *粗体*:粗体
    • _斜体_:斜体
    • ~~删除线~~:删除线
    • 代码（无语法）:无语法代码
    • 上标^脚本^:上标脚本
    • 下表,,脚本,,:下标脚本
  • 链接
    • [[链接]]:链接
    • [[链接|链接描述]]或者[[链接源地址][链接描述]]:个人导航
    • 纯链接：http://blog.tinyxd.me
  • 标题
    • = 1级标题 =
    • == 2级标题 ==
  • 段落

段落间以一个空行分割。

  • 列表 
    
* 符号列表1
* 符号列表2
* 符号列表3
  * 符号列表
  * 符号列表
    - 符号列表
    - 符号列表
      # 符号列表
      # 符号列表   
        列表第一行
        列表第二行
      # 符号列表
  • 表格 
    | 年   | 最低温度 | 最高温度 |
|------|----------|----------|
| 2001 | 10       | 20       |
| 2002 | 20       | 30       |

效果如下：

  • 预格式化文本
    年 最低温度 最高温度
    2001 10 20
    2002 20 30
  • 块级引用
  • 注释
